  vim: set sw=1 ts=1 sts=0 et nonu nornu nowrap tw=40 foldlevel=1 foldmethod=indent foldtext=getline(v\:foldstart) nospell:

  Use \? to open this file in Vim.

  NOTATION:

  \      Leader
  ‹n›    A (possibly optional) number
  ‹c›    A single character
  ‹r›    A register (", *, 0, 1, …)
  ‹Move› A motion (e.g., 4jll) or a text
         object (iw, ap, …)
  ‹P›    A regular expression (pattern)
  ‹Cmd›  An Ex command
  ‹Path› A file path or a file name
  ‹Var›  A variable or option name
  «C-…»  Press ctrl and … together
  «Spc»  Press space bar

  Similarly, other special keys are
  written between « and ». All other
  characters are meant to be typed
  literally.

  Notation used in the comments:

  C    Type in Command (Ex) mode
  I    Type in Insert mode
  V    Type in Visual mode
  M    Motion (may be used with an
       operator)
  O    Operator (use in Visual mode, or
       before a motion or a text object)
  T    Text object (use in Visual mode
       or after an operator)
  *    User-defined mapping
  »    Forwards/rightwards
  «    Backwards/leftwards
  WORD Space-delimited word
