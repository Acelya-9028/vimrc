---
  About
  vim: set sw=1 ts=1 sts=0 et nonu nornu nowrap foldlevel=1 foldmethod=indent foldtext=getline(v\:foldstart) nospell:

  Use ,cs to open this file in Vim.

  <n> denotes a mandatory number.
  [n] denotes an optional number.
  <c> denotes a single character.
  [p] denotes a pattern.
  [move] denotes a move command.
  Other placeholders should be clear
  enough from the context.

  Every user-defined command either
  starts with a comma or is a function
  key. The only exception is ctrl-j,
  which is mapped to Esc in insert and
  visual mode for convenience.

  This cheat sheet is roughly
  organized after TextWrangler's menu
  structure, because TextWrangler is
  the editor I am most familiar with.

  Note that this not a replacement
  for a manual. The comments are
  (overly) simplified: refer to Vim's
  help for all the details.
---
 File
  Basic
  :e <path>       - New/open file
  :[n]bd          - Close file
  ,w
  :w              - Save file
  :w!             - Save RO file
  :wq             - Save and quit
  :wqa            - Save all and quit
  :q              - Quit
  :q!             - Quit without saving
  :qa!            - Quit all without saving
 :mks            - Save session (restore with vim -S)
 ,l              - List open files
 :ls
 :[n]b           - Edit file n
 ,1              - Edit file 1
 …               …
 ,0              - Edit file 10
 ,f              - Edit next file
 ,b              - Edit prev file
 ,g              - Edit last used file

 Edit
  Basic
  <esc>         - Return to normal mode
  ctrl-[
  i             - Insert at cursor
  a             - Append after the cursor
  I             - Insert at the start of line
  A             - Append at the end of line
  o             - Open new line below
  O             - Open new line above
  r <c>         - Replace char
  R             - Replace mode (overwrite)
  .             - Repeat last command
 u               - Undo
 ctrl-r          - Redo
 d               - Cut selection
 d [move]        - Cut to the move-to point
 [n]dd           - Cut line(s)
 di <c>          - Cut between matching pairs of <c>
 [n]x            - Cut char(s)
 [n]X            - Cut prev char(s)
 "*d             - Cut into system clipboard
 y               - Copy selection
 y [move]        - Copy to the move-to point
 yy              - Copy line
 "*y             - Copy to system clipboard
 p               - Paste after cursor
 P               - Paste before cursor
 ]p              - Paste below and align with current line
 ]P              - Paste above and align with current line
 "*p             - Paste from system clipboard
 c               - Replace selection
 c [move]        - Replace up to the move-to point
 cc              - Replace line
 ci <c>          - Replace between matching pair of <c>
 v               - Visual mode (selection)
 V               - Linewise visual mode
 ctrl-v          - Visual block mode
 ctrl-v [move] I - Insert in multiple lines
 ctrl-v [move] A - Append to multiple lines
 {v}             - Select paragraph
 gv              - Reselect last selection
 ,a              - Select all
 ,ts             - Remove trailing space
 J               - Join with next line

 Text
 ~               - Change case
 U               - Change selection to uppercase
 u               - Change selection to lowercase
 gU [move]       - Change to uppercase
 gu [move]       - Change to lowercase
 gUU             - Change line to uppercase
 guu             - Change line to lowercase
 ,U              - Capitalize selected words
 ctrl-n          - Autocomplete
 <               - Shift selection to the left
 >               - Shift selection to the right
 <%              - Shift block to the left
 ==              - Fix line indent
 >%              - Shift block to the right
 ,a=             - Reindent all (C-style)
 :set tw=<n>     - Set text width (0 = window width)
 gw              - Hard-wrap selection at text width
 gw [move]       - Wrap text up to move-to point
 gww             - Wrap text in current line
 ,r              - Hard-wrap paragraph
 ,t              - Toggle hard/soft tabs
 ,[              - Decrease tab width
 ,]              - Increase tab width
 :retab          - Detab (needs expandtab on - see ,t)
 :retab!         - Entab (needs expandtab off - see ,t)
 ,s              - Toggle spell checking
 ]s              - Go to next mispelled word
 [s              - Go to prev mispelled word
 z=              - Suggest corrections
 ga              - Character info for char under cursor

 View
  Scrolling
  ctrl-f        - Scroll down a page
  ctrl-b        - Scroll up a page
  ctrl-d        - Scroll down half a page
  ctrl-u        - Scroll up half a page
  ctrl-e        - Scroll down a bit (two lines)
  ctrl-y        - Scroll up a bit (two lines)
  zl            - Scroll right
  zh            - Scroll left
 :set wrap       - Soft-wrap text
 :set nowrap     - Do not wrap text
 ,n              - Toggle line numbers
 ,m              - Toggle relative line numbers
 ,i              - Toggle invisibles
 zo              - Fold
 zc              - Unfold
 zM              - Fold all
 zR              - Unfold all
 <F7>            - Toggle background color
 ,F              - Toggle distraction-free mode
 :set cc=<n>     - Page guide at column n
 :set cc=""      - No page guide

 Search
 /[p]            - Search forward
 ?[p]            - Search backward
 n               - Find next (same direction)
 N               - Find prev (opposite direction)
 [cmd] gn        - Select next match (and apply cmd)
 [cmd] gN        - Select prev match (and apply cmd)
 ,h              - Toggle highlighting
 :s/[p]/[p]/     - Find and replace in selection
 :%s/[p]/[p]/g   - Find and replace all

 Go
  Basic movement
  [n]h          - Move left
  [n]←
  [n]l          - Move right
  [n]→
  [n]k          - Move up
  [n]↑
  [n]j          - Move down
  [n]↓
  [n]gj         - Move down i soft-wrapped lines
  [n]gk         - Move up in soft-wrapped lines
  [n]w          - Start of next punctuation-separated word
  [n]b          - Start of prev punctuation-separated word
  [n]e          - End of next punctuation-separated word
  [n]ge         - End of prev punctuation-separated word
  [n]W          - Start of next space-separated word
  [n]B          - Start of prev space-separated word
  [n]E          - End of next space-separated word
  [n]gE         - End of prev space-separated word

  $             - End of line
  ^             - First non-blank char of line
  [n]}          - Next blank line
  [n]{          - Prev blank line
  gg            - First line
  G             - Last line
 :[n]            - Go to line…
 [n]G

 *               - Forward search for word under cursor
 #               - Backward search for word under cursor
 [n]f <c>        - Find next (nth) <c> in current line
 [n]F <c>        - Find prev (nth) <c> in current line
 %               - Jump between matching () or {}
 ctrl-o          - Go to older cursor position
 ctrl-i          - Go to newer cursor position
 ctrl-]          - Follow link
 ctrl-T          - Back to prev link
 ma              - Set mark called 'a'
 `a              - Go to mark 'a'
 mb              - Set mark called 'b'
 `b              - Go to mark 'b'
 …               …
 mA              - Set global mark 'A'
 `A              - Go to global mark 'A'
