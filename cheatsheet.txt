---
  About
  vim: set sw=1 ts=1 sts=0 et nonu nornu nowrap foldlevel=1 foldmethod=indent foldtext=getline(v\:foldstart) nospell:

  Use ,cs (press comma, then c, then s)
  to open this file in Vim.

  This cheat sheet is roughly
  organized after TextWrangler's menu
  structure, because TextWrangler is
  the editor I am most familiar with.
  Looks nice when viewed with Monaco.

  Note that this not a replacement
  for a manual. The comments are often
  (overly) simplified: refer to Vim's
  help pages for all the details.

  NOTATION:

  ⓝ      denotes a (possibly optional)
         number.

  ⓒ      denotes a single character.

  ‹P›    denotes a pattern (i.e., a
         regular expression).

  ‹Cmd›  denotes a command (e.g., d to
         delete).

  ‹Move› denotes an arbitrary move
         command (e.g., 4jll).

  ‹Path› denotes a file path or a
         file name.

  ‹Reg›  denotes a register (", *, …).

  ‹Var›  denotes a variable name.

  «C-…» means the ctrl character must
  be pressed along with what follows.
  A space after this key combination
  must not be typed literally!

  Similarly, other special keys are
  written between « and ».

  All other characters are meant to be
  typed literally.

 File
  Basic
  :e ‹Path›      - New/open file
  :ⓝ bd          - Close file
  ,w
  :w             - Save file
  :w!            - Save read-only file
  :wq            - Save and quit
  :wqa           - Save all and quit
  :q             - Quit
  :q!            - Quit without saving
  :qa!           - Quit all without saving
 :mks          - Save session
                 (restore with vim -S)
 ,l            - List open files
 :ls
 :ⓝ b          - Edit file n
 ,1            - Edit file 1
 …             …
 ,0            - Edit file 10
 ,f            - Edit next file
 ,b            - Edit prev file
 ,g            - Edit last used file

 Edit
  Basic
  «Esc»        - Return to normal mode
  «C-[»
  i            - Insert at cursor
  a            - Append after the cursor
  I            - Insert at the start of line
  A            - Append at the end of line
  o            - Open new line below
  O            - Open new line above
  rⓒ           - Replace char
 ,1            - Edit file 1
 …             …
 ,0            - Edit file 10
 R             - Replace mode (overwrite)
 .             - Repeat last command
 u             - Undo
 «C-r»         - Redo
 d             - Cut selection
 d‹Move›       - Cut to the move-to point
 ⓝ dd          - Cut line(s)
 0D            - Clear line
 diⓒ           - Cut between matching pairs of chars
 ⓝ x           - Cut char(s)
 ⓝ X           - Cut prev char(s)
 "*d           - Cut into system clipboard
 y             - Copy selection
 y‹Move›       - Copy to the move-to point
 yy            - Copy line
 "*y           - Copy to system clipboard
 p             - Paste after cursor
 P             - Paste before cursor
 ]p            - Paste below and align with current line
 ]P            - Paste above and align with current line
 "*p           - Paste from system clipboard
 c             - Replace selection
 c‹Move›       - Replace up to the move-to point
 cc            - Replace line
 ci ⓒ          - Replace between matching pair of chars
 v             - Visual mode (selection)
 V             - Linewise visual mode
 «C-v»         - Visual block mode
 «C-v»‹Move›I  - Insert in multiple lines
 «C-v»‹Move›A  - Append to multiple lines
 «C-v»‹Move›x  - Delete char in multiple lines
 {v}           - Select paragraph
 gv            - Reselect last selection
 ,a            - Select all
 ,ts           - Remove trailing space
 J             - Join with next line

 ga            - Character info for char under cursor
